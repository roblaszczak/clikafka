# CLIkafka

Simple CLI tool for Kafka written in GO (golang).

## Installation

Just run

    go get github.com/roblaszczak/clikafka


## Usage

Simplest usage:

    clikafka my_pro_topic --broker=my-kafka-server.com:9092

For available options run

    clikafka --help
